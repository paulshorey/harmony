Instead of trying to rewrite CSS in JS, or just using classNames for everything, let <a>CSS and JS</a> to work in harmony, each doing what they do best. This library uses a very small global stylesheet to define colors and lights/darks. Because CSS is "**cascading**", every element knows if its on dark or light, and can adjust automatically.
