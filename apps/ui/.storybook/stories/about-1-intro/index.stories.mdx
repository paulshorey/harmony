Instead of trying to rewrite CSS in JS, or just use classNames for everything, CSS and JS can work together, helping each to do what each does best. This library uses a very small global stylesheet to define color groups (for branding) and light/dark versions of each. Because CSS is <b>cascading</b>, every element will know if its on dark or light, and will adjust itself automatically.
