<Block as="h2" ss="margin-top:2rem;">
  <Block as="span" textgradient="rainbow">
    For better DX and UX
  </Block>
</Block>

Instead of trying to rewrite CSS in JS, or just using classNames for everything, let <a>CSS and JS</a> to work in harmony, each doing what they do best. This library uses a very small global stylesheet only to manage colors and lights/darks. Because of CSS "**cascading**" feature, each element knows if its on dark or light, and can adjust its colors automatically. No need to manually shade each element.
